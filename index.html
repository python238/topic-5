Довольно часто вам нужно перебрать с помощью цикла все числа от одного до другого, например от 0 до 5. С помощью цикла с условием вы можете сделать это примерно так:

x = 0

while x < 5:

print(5)

x = x + 1

для этой частой операции существует другая конструкция: цикл for in range

for x in range(5):

print(x)

обе эти программы напечатают

0

1

2

3

4

На самом деле цикл for in можно использовать не только с range, однако это мы рассмотрим на дальнейших занятиях.

У range есть три формы:

# от 0 до заданного числа

range(до)

# от заданного числа до заданного числа

range(от, до)

# от заданного числа до заданного числа с фиксированным шагом

range(от, до, шаг)

Например

range(10, 25, 3) даст значения

10

13

16

19

22

Конец диапазона (число “до”) не входит в диапазон, если вы хотите включить его, то вам нужно увеличить это число на один. Это может показаться неудобным, однако потом не включение правой границы в диапазон будет удобно использоваться.

Если конец диапазона равен его началу или больше его, то такой диапазон будет пустым -- цикл по нему не сделает ни единой итерации.

Так

for x in range(a,b,s):

print(x)

эквивалентно

x = a

while x < b:

print(x)

x = x + s

Также шаг в range может быть отрицательным, в таком случае число число “от” должно быть больше числа “до”, и числа буду перебирать от больших к меньшим. Например

range(3, -3, -1) даст значения 3, 2, 1, 0, -1 и -2

range(20, 10, -3) даст значения 20, 17, 14 и 11
